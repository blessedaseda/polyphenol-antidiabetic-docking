# polyphenol-antidiabetic-docking
Computational validation of selected polyphenols as potential antidiabetic agents through molecular docking against alpha-amylase (PDB: 2QV4)

Project Overview

This project investigates the antidiabetic potential of selected dietary polyphenols using molecular docking against human alpha-amylase (PDB ID: 2QV4).

The study compares natural polyphenolic compounds with:

* Acarbose (standard alpha-amylase inhibitor)
* Maltotriose (substrate molecule)

to evaluate inhibitory activity based on:

* Docking scores
* Residue interaction analysis
* Binding-mode analysis

## Objective

To computationally validate whether selected polyphenols can act as alpha-amylase inhibitors and potentially serve as lead compounds for antidiabetic drug discovery.

## Selected Compounds

| Acarbose     | Standard Drug | -9.242                   |
| Quercetin    | Polyphenol    | -9.134                   |
| Luteolin     | Polyphenol    | -9.099                   |
| Catechin     | Polyphenol    | -8.773                   |
| Pelargonidin | Polyphenol    | -8.631                   |
| EGCG         | Polyphenol    | -8.498                   |
| Maltotriose  | Substrate     | -6.948                   |



## Key Findings

* Quercetin and Luteolin showed binding affinities highly comparable to Acarbose.
* Strong inhibition correlated with:

  * Conventional hydrogen bonding
  * Pi-pi stacking interactions
  * Pi-anion interactions
  * Hydrophobic stabilization
* Maltotriose exhibited the weakest binding affinity, consistent with its role as a substrate rather than an inhibitor.
